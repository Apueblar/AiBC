IT Applications in Business and Commerce — AiBC

IBM AI Racing League · Country Challenge 2026 · TORCS autonomous driver (Python)

Autonomous racing in The Open Racing Car Simulator, aligned with the course project board ITAPP_Tue1115_AiBC: research, implementation, measurable laps, and continuous integration.

Objective

Program an AI driver for TORCS; qualification focuses on lap performance on the Corkscrew track. The competition framing matches real delivery practice: build → test → explain → defend the strategy.

Stack referenced in-repo: Python client over the SCR/snakeoil interface, with optional path toward IBM Granite for later stages.

Delivered so far

  • Team & research: TORCS SCR client workflow and how to race via the simulator.
  • Sensor layer: parsing and normalization of TORCS sensors, derived features, CSV logging for analysis (sensor_processor.py, results/sensors/).
  • Driver: rule-based Alpha driver (v0.2) — adaptive speed from rangefinders, steer smoothing and corner mode, automatic gearing, traction-aware throttle/brake (car_agent.py).
  • Runner: automated TORCS menu navigation and UDP session loop (run.py).
  • DevOps: GitHub Actions workflow running the stack in Docker, capturing lap time, distance, reward; appending runs to results/history.csv.
