# AiBC
AiBC stands for:

IBM AI Racing League - Country Challenge 2026

The IBM AI Racing League is a global IBM competition in which student teams build an autonomous AI driver. Participants programme the car in the TORCS (The Open Race Car Simulator) simulator using Python. The AI engine is IBM Granite. The goal of the qualification round is to achieve the best lap time on the Corkscrew track.

This is not just an ‘AI race,’ but a simulation of the realities of the industry: build → test → explain → defend your strategy.
